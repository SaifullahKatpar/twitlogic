TwitLogic: real-time #SemanticWeb in <= 140 chars


Instant publication, coupled with the rise of geosocial networking, presents new opportunities for real-time and location-based services. Much as these services are changing the nature of search on the World Wide Web, so the Semantic Web will be both challenged and enhanced by real-time content. TwitLogic is a semantic data aggregator which brings together a collection of compact formats for structured microblog content with Semantic Web vocabularies and best practices in order to augment the Semantic Web with real-time, user-driven data.


This Web interface is a work in progress. For now, you can browse the knowledge base in Zitgist. Or if you're feeling adventurous, you can explore the knowledge base with curl, like so: 
curl -L -H "Accept: application/x-trig" http://twitlogic.fortytwo.net/hashtag/sdow2009
 This gives you access to additional Named Graph information, allowing you to follow the nanotation provenance trail.
